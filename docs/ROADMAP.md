# Home Management App - Roadmap

## Current Features (Completed)

### Core
- [x] Google OAuth authentication
- [x] Google Calendar integration (view, create, edit, delete events)
- [x] External calendar sync (iCal URLs)
- [x] Contacts (synced from Google, tags, favorites)
- [x] Weather widget

### Household
- [x] Shopping lists (grocery + other items, favorites)
- [x] Recipes (manual entry, URL import, AI suggestions)
- [x] Home assets inventory
- [x] Notes board

### Entertainment
- [x] Movies collection (OMDb/TMDB integration, ratings)
- [x] Games collection (loan tracking)

### Family
- [x] Kids rewards (sticker system)
- [x] ADHD-friendly task manager (energy levels, AI task breakdown)

### Kiosk
- [x] Kiosk mode for touchscreen display
- [x] Smart input (AI-powered natural language processing)

## Roadmap

### Phase 1: Household Management Enhancements

#### Chores & Duties System
- [ ] Chore definitions with frequency (daily, weekly, monthly)
- [ ] Family member assignment and rotation
- [ ] Chore completion tracking and history
- [ ] Integration with Kids Rewards (earn stickers for chores)
- [ ] Visual chore chart for kiosk mode

#### Home Maintenance Reminders
- [ ] Maintenance task templates (HVAC filter, smoke detectors, etc.)
- [ ] Scheduled reminders with calendar integration
- [ ] Maintenance history log
- [ ] Cost tracking for maintenance items
- [ ] Seasonal maintenance checklists

#### Pantry & Freezer Inventory
- [ ] Item tracking with quantities
- [ ] Expiration date monitoring
- [ ] Low stock alerts
- [ ] Auto-add to shopping list when low
- [ ] Barcode scanning support (future)

#### Bill & Subscription Tracking
- [ ] Recurring bills with due dates
- [ ] Payment status tracking
- [ ] Calendar integration for due date reminders
- [ ] Monthly/yearly expense summary
- [ ] Subscription management (streaming services, etc.)

### Phase 2: Smart Home & Tech Integration

#### Notifications System
- [ ] Push notifications (web push API)
- [ ] Email notifications for important events
- [ ] Customizable notification preferences
- [ ] Daily digest option
- [ ] Urgent alerts (overdue bills, expiring items)

#### Mobile Companion App
- [ ] React Native or PWA mobile app
- [ ] Quick add items (shopping, tasks)
- [ ] Barcode scanning for pantry
- [ ] Push notifications on mobile
- [ ] Offline support with sync

#### Voice Assistant Integration
- [ ] Google Assistant actions
- [ ] "Add milk to shopping list"
- [ ] "What's on the calendar today?"
- [ ] "Start a 25-minute focus timer"
- [ ] Local voice commands via Raspberry Pi microphone (optional)

#### Smart Device Integration
- [ ] Home Assistant integration
- [ ] Display smart home device status on kiosk
- [ ] Control lights/thermostat from app
- [ ] Automation triggers (e.g., "leaving home" mode)

#### Dashboard Widgets
- [ ] Customizable kiosk dashboard layout
- [ ] Traffic/commute widget
- [ ] Package tracking widget
- [ ] Family calendar overview widget
- [ ] Quick actions bar

### Phase 3: Future Enhancements

#### Family Profiles
- [ ] Individual family member accounts
- [ ] Personal task lists and preferences
- [ ] Profile-specific calendar views
- [ ] Activity tracking per member

#### Meal Planning
- [ ] Weekly meal planner
- [ ] Auto-generate shopping list from meal plan
- [ ] Recipe suggestions based on pantry
- [ ] Nutritional tracking (optional)

#### Vehicle Maintenance
- [ ] Vehicle profiles (make, model, year)
- [ ] Maintenance schedule tracking
- [ ] Mileage logging
- [ ] Service history and costs

#### Photo Frame Mode
- [ ] Slideshow from Google Photos or local folder
- [ ] Idle screen saver for kiosk
- [ ] Family photo rotation
